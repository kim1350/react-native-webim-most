# react-native-webim-most

Webim chat wrapper for React-Native. Supports Android and iOS

## Installation

```sh
npm install react-native-webim-most
```

## Usage


```js
import { multiply } from 'react-native-webim-most';

// ...

const result = await multiply(3, 7);
```
